This project was written by Rony Raju as an exercise given by Wolfram for a job application.
It runs on a MySQL server out of port 3306 with the backend being a spring boot application witha maven wrapper that runs on port 8080.
The front end is react, and runs out of port 3000.

To run the app, start the java application while the MySQL server is running, and run the build of the react front end use command serve -s build.
All of the relevant database schema have been exported to the dbExport directory.
